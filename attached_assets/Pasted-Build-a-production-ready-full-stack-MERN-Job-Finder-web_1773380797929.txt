Build a production-ready, full-stack MERN Job Finder web application named “HireSync AI” for students and job seekers. The platform must connect to real-time job APIs, analyze uploaded resumes, recommend matching jobs, track applications, and notify users when relevant jobs are posted. The application must be modern, scalable, secure, mobile responsive, and portfolio-worthy.

TECH STACK
- Frontend: React.js + Vite
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Authentication: JWT + refresh tokens + bcrypt
- State management: Redux Toolkit or Context API
- UI: Tailwind CSS + shadcn/ui or clean custom component system
- File upload: Multer or cloud storage integration
- Notifications: Email + in-app + optional browser push
- Scheduling: Node cron / BullMQ for background job sync and alerts
- Resume parsing: NLP-based extraction from PDF/DOCX resumes
- AI matching engine: semantic skill matching + keyword scoring + role fit analysis
- Charts: Recharts
- Deployment-ready: frontend on Vercel, backend on Vercel, DB on MongoDB Atlas
